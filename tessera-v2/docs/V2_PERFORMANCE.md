Monitor tracks: FPS/current+average, worst frame ms, hitch count (>33ms), particle/filament count, quality, DPR, drawing buffer size, preset.

Use `P` to toggle monitor, `1..4` to shift quality.

Common stutter causes: oversized DPR on high resolution screens, filament budgets too high, too many simultaneous browser tabs/GPU contention.

Debug: `tesseraDebug()`.
