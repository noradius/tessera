# Tessera V2 Visual Protection

## Intentionally protected
- Deep black-blue ambient ground with volumetric fade.
- Spectral particulate field (not starfield/HUD).
- Temporary white filaments from proximity relations.
- Rest state that remains alive even with minimal texture values.
- Uncertainty rendered as fog/softness, not explicit confidence text.

## Intentionally avoided
- Dashboard framing.
- Metrics and scoring UI.
- Emotion language and anthropomorphic claims.
- Reward-like overreaction to user activity.

## Current limitations
- This prototype still uses a simplified relation-sampling strategy for filaments.
- Arrival is manually preset-triggered; no detector logic is included in sandbox.
- A deeper boid neighborhood solver from extension code can be ported in a subsequent pass.
