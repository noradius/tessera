# Tessera V2.1 — Visual DNA Restoration

## Living mechanisms in legacy visual lineage
- Persistent per-particle hue identity with broad spectral distribution and restrained luminance.
- Deep-water volumetric glow (layered point core + halo + atmospheric dampening).
- Localized boid-like neighbor dynamics via spatial hashing (alignment, cohesion, separation).
- Cymatic field attractors that periodically form structure and dissolve under changing texture weather.
- Contact filaments from relational proximity/resonance rather than random fixed pair sampling.
- Embodied event phases for Arrival (compress → hold → bloom → residue) and Heart Memory as a wave.
- Soft field boundary/falloff rather than hard clipping.

## Missing/reduced in pre-pass sandbox-v2
- Narrow teal hue band and weak persistent spectral individuality.
- Mostly central-attractor + sine drift motion (little neighbor-relational behavior).
- Filaments generated from sampled index pairs, graph-like risk.
- Reduced field topology/cymatic behavior.
- Arrival/Heart Memory mainly as parameter values, not strongly embodied temporal phases.
- Flat sparkle tendency and limited medium-depth atmosphere.

## Ported in this pass
- Reintroduced broad persistent spectral identities per particle with restrained rest-state brightness.
- Added layered particle shader behavior (core + halo + atmospheric tint) for deep-water luminosity.
- Added spatial-grid neighborhood solver with alignment/cohesion/separation for murmuration motion.
- Added cymatic-style field forcing coupled to active textures (reach/folding/response).
- Replaced filament sampling with proximity + hue-resonance contact traces (temporary, thin, white).
- Added Arrival phase embodiment and Heart Memory wave carried through state and visual uniforms.
- Switched boundary behavior to soft radial porthole containment/falloff.

## Intentionally left for later
- No extension integration yet.
- No API coupling/detector inference layer for autonomous event triggering.
- No dashboard/product UI layer.
- No full parity port of every legacy micro-effect; this is a restoration pass, not final convergence.
