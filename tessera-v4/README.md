# Tessera V4
A standalone conversation-weather instrument. It witnesses conversational texture (drift, tracking, friction, arrival, residue) without scoring, diagnosing, or classifying emotion.

## Not
- not sentiment analysis
- not quality scoring
- not human/AI interiority detector

## Run
`python -m http.server 8000` then open `http://localhost:8000/tessera-v4/index.html`.

## Controls
Preset buttons, recipe selector, quality 1-4, pause, perf overlay, capture, copy debug/settings.
Keyboard: R/T/F/A/H/M/S/D/U, P, Space, C, 1/2/3/4.

## Debug
Run `tesseraDebug()` in console for runtime truth state.
