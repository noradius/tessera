# Tessera Lab
Tessera Lab is a standalone vertical slice: an ambient living conversation weather instrument for exploring motion grammar, texture choreography, and extension-ready architecture.

## Not
Not emotion detection, not sentiment scoring, not a dashboard.

## Run
`py -m http.server 8000` then open `http://localhost:8000/tessera-lab/index.html`.

## Controls
Preset buttons, recipe selector, intensity/motion/spectral/filament/event/density sliders, pause/capture/copy debug/slow-mo.

## Keyboard
R/T/F/A/H/M/S/D/U presets; P performance overlay; Space pause; C capture; 1/2/3/4 quality.

## Debug
Use `window.tesseraDebug()`.

## Quality
Low/Medium/High/Ultra with particle counts up to ~14k.
